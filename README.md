# Trevor Nanney — Portfolio

Modern, single‑page portfolio site for trevornanney.net.

## Tech
- HTML / CSS / JS (static)
- Netlify Forms
- Google Analytics 4

## Local preview
```
python3 -m http.server 5173
```
Then open `http://localhost:5173`.

## Deploy
This repo is connected to Netlify. Pushes to `main` deploy automatically.

## Notes
- Contact form posts to `/success` (see `success.html`).
- Update the GA4 ID in `index.html` if needed.
